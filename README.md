# LightInkMsgPack
C++ for MsgPack from project LightInk


#build


##win32

* install vs2012
* open file ./windows/LightInkMsgPack.sln
* build
* the file output in ./bin/Debug(Release)/

##linux

* install cmake
* mkdir ./Linux/build
* cd ./Linux/build
* cmake ../
* make
* the file output in ./bin/Debug(Release)/
